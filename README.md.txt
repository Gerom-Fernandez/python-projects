# 🐍 Python Projects  

Este repositorio contiene mis **ejercicios y proyectos en Python**, desarrollados durante mi aprendizaje y práctica personal.  
Aquí muestro ejemplos de lógica de programación, estructuras de datos, POO, y algunos proyectos propios que voy ampliando poco a poco.  

---

## 📂 Estructura del repositorio  

- **Ejercicios de Python/**  
  Programas básicos de práctica para aprender los fundamentos de Python:  
  - Calculadora básica  
  - Promedio de una lista  
  - Clasificación de números  
  - Registro de datos personales  
  - Uso de funciones y módulos personalizados  

- **Import y Export/**  
  Ejercicios para practicar la importación y exportación de módulos en Python.  

- **Proyecto_personales/**  
  Proyectos propios más completos, como:  
  - 🛒 Carro de compra  
  - 💬 Sistema de mensajes de WhatsApp (simulación)  
  - 🃏 Sistema de cartas (modo básico y pro)  

- Archivos sueltos: ejemplos de **POO (clases, herencia, encapsulamiento)** y manejo de estructuras (`listas`, `tuplas`, `diccionarios`).
